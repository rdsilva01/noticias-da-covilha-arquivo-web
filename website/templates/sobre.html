Sobre o Projeto

O projeto Arquivo NC - Notícias da Covilhã @ Arquivo Web é uma proposta de projeto desenvolvida por Rodrigo Silva no âmbito da Unidade Curricular de Projeto da Licenciatura em Engenharia Informática, com orientação do professor Ricardo Campos (ricardo.campos@ubi.pt).

O objetivo principal deste projeto é preservar a história e o legado do jornal "Notícias da Covilhã" tornando o seu conteúdo histórico facilmente acessível ao público e investigadores. Além disso, o projeto visa criar um sistema de pesquisa para recuperar informações a partir dos dados preservados pelo Arquivo.pt.

Work In Progress
O plano de trabalho incluiu várias etapas, desde a definição e planeamento até à documentação e apresentação dos resultados. Os requisitos técnicos/académicos abrangem conhecimentos em Python, linguagens de programação web (Flask, HTML, CSS e JS), Elasticsearch, Docker, entre outros.
